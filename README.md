# How to use
 1. Add this to your package's pubspec.yaml file:

````dart
dependencies:
  flutter_number_animation: ^0.0.1
  ````
 
 2. Select your favorite loader and add it as a widget!
 you can customise it however you like!
 
 ````dart
 FlutterLoaders.pulse(
              Icon(
                Icons.android,
                color: Colors.green,
                size: 40.0,
              ),
              Colors.green)
  ````
